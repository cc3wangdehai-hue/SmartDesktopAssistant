## 两种运行模式

### 模式1：独立小组件（推荐新手）

**选择：1**

特点：
- 界面简洁，两个Widget并排显示
- 适合快速测试Widget功能
- 窗口可拖动、调整大小
- 无法置顶显示

**选择：2**

特点：
- 专业界面，带标题栏和系统托盘
- Widget可置顶显示在桌面
- 支持隐藏到托盘后台运行
- 窗口位置自动保存

## 天气Widget测试

### 测试1：检查初始显示

1. 观察界面显示
2. 确认显示两个城市（默认：北京、上海）
3. 确认显示温度数字
4. 确认天气描述正确

**预期结果**：界面显示两个天气卡片，绿色和蓝色渐变背景

### 测试2：手动刷新

1. 找到第一个城市的刷新按钮（圆形按钮带⟳）
2. 点击该按钮

**预期结果**：
- 显示"加载中..."
- 几秒后更新为新数据
- 如果网络失败显示"加载失败"并可重试

### 测试3：验证天气描述

天气状况使用中文显示：
- 0 = 晴
- 1 = 晴间多云
- 2 = 多云
- 3 = 阴
- 61/63/65 = 雨
- 71/73/75 = 雪
- 其他请对照实际显示

### 测试4：观察自动刷新

等待30分钟，观察天气是否自动更新

## TODO Widget测试

**预期结果**：

1. 点击待办右侧的 × 按钮

**预期结果**：
- 该待办从列表中移除


## 常见问题

### Q1：提示".NET SDK未安装"

**解决方法**：
1. 访问 https://dotnet.microsoft.com/download/dotnet/8.0
2. 下载并安装 **SDK**（不是Runtime）
3. 重启电脑
4. 再次运行 `run.bat`

### Q2：天气显示"加载失败"

**可能原因**：
- 网络连接问题
- API临时不可用

**解决方法**：
- 检查网络连接
- 等待几分钟后重试
- 点击"点击重试"文字

### Q3：托盘图标不显示（仅模式2）

**解决方法**：
- 确认程序正在运行（任务管理器中查看）
- 可能在通知区域的隐藏图标中，点击"<"展开

### Q4：TODO数据丢失

**可能原因**：
- 手动删除了数据文件
- 软件异常退出

**数据文件位置**：
```
%LOCALAPPDATA%\SmartDesktopAssistant\todos.json
```

### Q5：选择模式后程序无响应

**解决方法**：
- 按 Ctrl+C 终止当前程序
- 重新运行 run.bat

## 技术信息

### 项目结构

```

├── SmartDesktopAssistant/     # 主程序
│   ├── MainWindow.xaml.cs    # 主窗口代码

├── WidgetsStandalone/        # 独立测试版
│   └── MainWindow.xaml       # 测试窗口
├── Widgets/
│   ├── WeatherWidget/        # 天气组件库
│   │   ├── WeatherControl.xaml    # UI定义（360风格）
│   │   ├── WeatherControl.xaml.cs # 交互逻辑
│   │   └── WeatherService.cs      # Open-Meteo API
│   └── TodoWidget/           # 待办组件库
│       ├── TodoControl.xaml       # UI定义
│       ├── TodoControl.xaml.cs    # 交互逻辑
│       └── TodoStorage.cs         # 本地JSON存储
├── docs/
│   ├── 第一期技术方案.md
│   └── 用户测试指南.md      # 本文档
└── Output/                   # 编译输出目录
```

### 技术栈

- **.NET 8.0** - 运行时
- **WPF** - UI框架
- **ElementHost** - WinForms/WPF集成
- **Open-Meteo API** - 天气数据（免费无需API Key）
- **Newtonsoft.Json** - JSON序列化

### 天气API说明

使用 **Open-Meteo** 免费天气API：
- 官网：https://open-meteo.com/
- 特点：完全免费，无需注册，无需API Key
- 城市搜索API：https://geocoding-api.open-meteo.com/v1/search
- 天气数据API：https://api.open-meteo.com/v1/forecast
