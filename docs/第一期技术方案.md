## 一、项目架构概览

### 1.1 技术选型理由

| 技术方案 | 优势 | 劣势 |
|---------|------|------|
| Desktop Fences+ (WinForms) | 成熟的桌面图标管理，开源且活跃 | UI现代化程度较低 |
| Desktop Widgets Board (WPF) | 现代化UI，支持动画和数据绑定 | 独立运行，需适配透明窗口 |
| **ElementHost集成** | 微软官方方案，WinForms/WPF互通 | 需要处理透明背景和焦点问题 |

**最终方案**：以 Desktop Fences+ 为基础框架，通过 ElementHost 将 Desktop Widgets Board 的WPF控件嵌入其中。

```csharp
// NonActivatingWindow.cs 核心逻辑
public class NonActivatingWindow : Window
{
    private const int WS_EX_NOACTIVATE = 0x08000000;
    
    protected override void OnSourceInitialized(EventArgs e)
    {
        base.OnSourceInitialized(e);
        var hwndSource = (HwndSource)PresentationSource.FromVisual(this);
        hwndSource.AddHook(WndProc);
        SetWindowLong(... , GWL_EXSTYLE, ... | WS_EX_NOACTIVATE);
    }
    
    private IntPtr WndProc(IntPtr hwnd, int msg, IntPtr wParam, IntPtr lParam, ref bool handled)
    {
        if (msg == WM_MOUSEACTIVATE)
        {
            handled = true;
            return new IntPtr(MA_NOACTIVATE);  // 不激活窗口
        }
        return IntPtr.Zero;
    }
}
```

```csharp
// MainForm.cs
public class MainForm : Form
{
    private ElementHost _elementHost;
    
    private void InitializeElementHost()
    {
        // 1. 创建ElementHost控件
        _elementHost = new ElementHost
        {
            Size = new Size(350, 180),
            BackColor = Color.Transparent  // 关键：透明背景

        // 2. 创建WPF控件
        var weatherCard = new WeatherCardControl();
        
        // 3. 嵌入WPF控件
        _elementHost.Child = weatherCard;
        
        this.Controls.Add(_elementHost);
    }
}
```

### 3.3 透明背景处理

**问题**：ElementHost默认不支持透明背景

**解决方案**：

```csharp
// 方案1：设置透明色
_elementHost.BackColor = Color.Transparent;
_elementHost.TransparentBackground = true;

// 方案2：对于真正的透明窗口，使用以下API
[DllImport("user32.dll")]
static extern bool SetLayeredWindowAttributes(IntPtr hwnd, uint crKey, byte bAlpha, uint dwFlags);

const int LWA_COLORKEY = 0x1;
SetLayeredWindowAttributes(this.Handle, Color.Transparent.ToArgb(), 0, LWA_COLORKEY);
```

### 3.4 鼠标焦点处理

**问题**：WPF控件获取焦点后可能阻止桌面交互

**解决方案**：

```csharp
// 在WPF控件中处理
protected override void OnGotMouseCapture(MouseEventArgs e)
{
    base.OnGotMouseCapture(e);
    // 释放鼠标捕获
    this.ReleaseMouseCapture();
}
```
